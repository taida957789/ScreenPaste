# ScreenPaste

輕巧的 Windows 截圖與標註工具：一鍵截圖、即時標註、釘選到螢幕。

## 功能

### 截取
- 整個畫面（支援多螢幕）
- **視窗與 UI 元素自動辨識**：滑鼠移到視窗或介面元素（按鈕、面板、網頁區塊）上自動描出範圍，點一下即截取；**滾輪**可切換偵測層級（元素 ⇄ 視窗）
- 拖曳框選自訂範圍，附尺寸標示與放大鏡精準取邊

### 標註
截取後在滑鼠旁彈出**圖示工具列**（可拖曳移動、自動避開螢幕邊界）：

- **麥克筆** — 實心筆觸，可調粗細 / 顏色 / 透明度
- **螢光筆** — 半透明疊色，可調粗細 / 顏色 / 透明度
- **文字** — 可選字體 / 大小 / 顏色 / 樣式（正常、粗體、斜體、刪除線）
- **形狀** — 方形 / 圓角方形 / 圓形，外框或填滿，可調線條粗細與顏色
- **直線 / 箭頭** — 拖曳畫線，兩端可各自切換是否為箭頭，可調粗細與顏色；按住 `Shift` 吸附 45° 角
- **貼圖** — 貼上 PNG / JPEG / WebP 圖片，可拖曳移動、滾輪縮放
- **模糊** — 高斯模糊 / 馬賽克，可調模糊程度
- **色彩選擇器** — 支援 Hex 輸入、RGB 與透明度調整
- **復原 / 重做**

### 輸出
- 複製到剪貼簿
- 儲存為 PNG / JPG（另存或快速存到預設資料夾）
- **釘選到螢幕**：影像浮於最上層，可拖曳、滾輪縮放、右鍵選單；釘多張時 Esc 先關聚焦的那張、無聚焦則全關

### 區域錄影
- 專屬全域熱鍵（預設 **F2**）啟動：拖曳框選要錄製的範圍，再按一次熱鍵（或點停止鈕）結束
- 錄製時以紅框標示範圍，旁邊小工具列顯示計時與停止鈕
- 結束後開啟**編輯器**：循環預覽、拖曳時間軸把手**修剪頭尾**（`Space` 播放、`←→` 逐格、`I`/`O` 設起訖點），匯出時顯示進度
- 匯出格式 **GIF / MP4 / WebP** 於編輯器直接切換；也可於設定改為「錄完直接儲存」跳過編輯器
- 可錄製滑鼠游標、可設定影格率（10–30 fps）
- 由隨附的 `ffmpeg` 進行編碼，無需另外安裝

### 其他
- 全域熱鍵（截圖預設 **F1**、區域錄影預設 **F2**）與**系統匣**圖示啟動
- **多國語言**：繁體中文 / English / 日本語 / 한국어 / Français / Deutsch / Español（預設跟隨系統）
- **淺色 / 深色 / 跟隨系統**主題
- 集中式**設定視窗**：語言、熱鍵、主題、開機啟動、儲存資料夾（熱鍵欄位可直接按組合鍵設定）
- 可設定**開機自動啟動**
- **自動更新**：透過 GitHub Releases 檢查新版（可於設定開關、或手動立即檢查），有更新時詢問並一鍵下載安裝

## 安裝

到 [Releases](https://github.com/taida957789/ScreenPaste/releases) 下載：

- **`ScreenPaste-<版本>-setup.exe`** — 安裝版（免系統管理員權限，安裝於 `%APPDATA%\ScreenPaste`，含開始功能表捷徑與解除安裝程式）
- **`ScreenPaste-<版本>-win-x64-portable.zip`** — 免安裝綠色版

兩者皆為自我包含，**無需另外安裝 .NET Runtime**，並已隨附 `ffmpeg`（供區域錄影使用）。

## 快速使用

![工具列](screenshots/ui.png)

1. 啟動後常駐系統匣，按 **F1**（或雙擊系統匣圖示）開始截圖。
2. 滑鼠移到視窗自動描框、點擊截取，或拖曳框選自訂範圍。
3. 在彈出的工具列選工具、調整參數，於選區上標註。
4. 按 **複製 / 儲存 / 釘選** 輸出；`Esc` 可退回重新框選或取消。

錄影：按 **F2** 拖曳框選範圍即開始錄製，再按一次 **F2**（或點停止鈕）結束；接著在編輯器中預覽、修剪頭尾並選擇 GIF/MP4/WebP 匯出（設定中可改為錄完直接儲存）。

設定（熱鍵、工具預設、主題、開機啟動、錄影格式/影格率等）皆可於系統匣選單 →「設定…」的視窗中調整。

> 從原始碼建置時，先執行 `scripts/fetch-ffmpeg.ps1` 下載隨附的 `ffmpeg.exe`（CI 於發佈時會自動執行）。
