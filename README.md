# ScreenPaste

輕巧的 Windows 截圖與標註工具：一鍵截圖、即時標註、釘選到螢幕。

## 功能

### 截取
- 整個畫面（支援多螢幕）
- **視窗自動辨識**：滑鼠移到視窗上自動描出範圍，點一下即截取
- 拖曳框選自訂範圍，附尺寸標示與放大鏡精準取邊

### 標註
截取後在滑鼠旁彈出**圖示工具列**（可拖曳移動、自動避開螢幕邊界）：

- **麥克筆** — 實心筆觸，可調粗細 / 顏色 / 透明度
- **螢光筆** — 半透明疊色，可調粗細 / 顏色 / 透明度
- **文字** — 可選字體 / 大小 / 顏色 / 樣式（正常、粗體、斜體、刪除線）
- **形狀** — 方形 / 圓角方形 / 圓形，外框或填滿，可調線條粗細與顏色
- **貼圖** — 貼上 PNG / JPEG / WebP 圖片，可拖曳移動、滾輪縮放
- **模糊** — 高斯模糊 / 馬賽克，可調模糊程度
- **色彩選擇器** — 支援 Hex 輸入、RGB 與透明度調整
- **復原 / 重做**

### 輸出
- 複製到剪貼簿
- 儲存為 PNG / JPG（另存或快速存到預設資料夾）
- **釘選到螢幕**：影像浮於最上層，可拖曳、滾輪縮放、右鍵選單；釘多張時 Esc 先關聚焦的那張、無聚焦則全關

### 其他
- 全域熱鍵（預設 **F1**）與**系統匣**圖示啟動
- **多國語言**：繁體中文 / English / 日本語 / 한국어 / Français / Deutsch / Español（預設跟隨系統）
- **淺色 / 深色 / 跟隨系統**主題
- 集中式**設定視窗**：語言、熱鍵、主題、開機啟動、儲存資料夾（熱鍵欄位可直接按組合鍵設定）
- 可設定**開機自動啟動**

## 安裝

到 [Releases](https://github.com/taida957789/ScreenPaste/releases) 下載：

- **`ScreenPaste-<版本>-setup.exe`** — 安裝版（免系統管理員權限，安裝於 `%APPDATA%\ScreenPaste`，含開始功能表捷徑與解除安裝程式）
- **`ScreenPaste-<版本>-win-x64-portable.zip`** — 免安裝綠色版

兩者皆為自我包含，**無需另外安裝 .NET Runtime**。

## 快速使用

![工具列](screenshots/ui.png)

1. 啟動後常駐系統匣，按 **F1**（或雙擊系統匣圖示）開始截圖。
2. 滑鼠移到視窗自動描框、點擊截取，或拖曳框選自訂範圍。
3. 在彈出的工具列選工具、調整參數，於選區上標註。
4. 按 **複製 / 儲存 / 釘選** 輸出；`Esc` 可退回重新框選或取消。

設定（熱鍵、工具預設、主題、開機啟動等）皆可於系統匣選單 →「設定…」的視窗中調整。
