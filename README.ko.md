# ScreenPaste

가벼운 Windows 스크린샷·주석 도구. 한 번의 키로 캡처하고, 즉시 주석을 달고, 화면에 고정할 수 있습니다.

## 기능

### 캡처
- 전체 화면(다중 모니터 지원)
- **창 자동 인식**: 마우스를 창 위에 올리면 자동으로 범위를 그려 주고, 클릭하면 캡처
- 드래그로 원하는 영역 선택. 크기 표시와, 가장자리를 정확히 잡는 돋보기 제공

### 주석
캡처 후 커서 옆에 **아이콘 도구 모음**이 나타납니다(드래그로 이동 가능, 화면 가장자리를 자동으로 피함):

- **마커** — 실선 획, 굵기 / 색상 / 불투명도 조절 가능
- **형광펜** — 반투명 겹침 색상, 굵기 / 색상 / 불투명도 조절 가능
- **텍스트** — 글꼴 / 크기 / 색상 / 스타일(보통, 굵게, 기울임, 취소선) 선택 가능
- **도형** — 사각형 / 둥근 사각형 / 원, 외곽선 또는 채우기, 선 굵기와 색상 조절 가능
- **스티커** — PNG / JPEG / WebP 이미지 붙여넣기, 드래그로 이동, 휠로 크기 조절
- **흐림** — 가우시안 흐림 / 모자이크, 흐림 강도 조절 가능
- **색상 선택기** — Hex 입력, RGB, 불투명도 조절 지원
- **실행 취소 / 다시 실행**

### 출력
- 클립보드로 복사
- PNG / JPG로 저장(다른 이름으로 저장, 또는 기본 폴더로 빠른 저장)
- **화면에 고정**: 이미지를 맨 위에 띄우고, 드래그·휠로 크기 조절·오른쪽 클릭 메뉴 제공. 여러 개를 고정한 경우 Esc는 먼저 포커스된 것을 닫고, 포커스가 없으면 모두 닫습니다

### 기타
- 전역 단축키(기본 **F1**)와 **시스템 트레이** 아이콘에서 실행
- **다국어 지원**: 번체 중국어 / English / 日本語 / 한국어 / Français / Deutsch / Español (기본값은 시스템 따름)
- **밝게 / 어둡게 / 시스템 따름** 테마
- 통합 **설정 창**: 언어, 단축키, 테마, 시작 시 실행, 저장 폴더(단축키 항목은 키 조합을 누르기만 하면 설정됨)
- **시작 시 자동 실행** 설정 가능
- **자동 업데이트**: GitHub Releases를 통해 새 버전 확인(설정에서 켜고 끄거나 수동으로 즉시 확인). 업데이트가 있으면 물어본 뒤 한 번의 클릭으로 다운로드·설치

## 설치

[Releases](https://github.com/taida957789/ScreenPaste/releases)에서 다운로드:

- **`ScreenPaste-<버전>-setup.exe`** — 설치 버전(관리자 권한 불필요, `%APPDATA%\ScreenPaste`에 설치, 시작 메뉴 바로 가기와 제거 프로그램 포함)
- **`ScreenPaste-<버전>-win-x64-portable.zip`** — 설치가 필요 없는 포터블 버전

둘 다 자체 포함형이므로 **별도의 .NET Runtime 설치가 필요 없습니다**.

## 빠른 시작

![도구 모음](screenshots/ui.png)

1. 실행하면 시스템 트레이에 상주합니다. **F1**(또는 트레이 아이콘 더블 클릭)로 캡처를 시작합니다.
2. 창 위에 마우스를 올려 자동으로 테두리를 그린 뒤 클릭해 캡처하거나, 드래그로 원하는 영역을 선택합니다.
3. 나타난 도구 모음에서 도구를 선택하고 매개변수를 조정한 뒤 선택 영역에 주석을 답니다.
4. **복사 / 저장 / 고정**으로 출력합니다. `Esc`로 다시 선택으로 돌아가거나 취소할 수 있습니다.

설정(단축키, 도구 기본값, 테마, 시작 시 실행 등)은 모두 트레이 메뉴 →「설정…」창에서 조정할 수 있습니다.
