# ScreenPaste

가벼운 Windows 스크린샷·주석·화면 녹화 도구. 한 번의 키로 캡처하고, 즉시 주석을 달고, 화면에 고정하고, 원하는 영역을 GIF / MP4 / WebP로 녹화하세요.

## 기능

### 캡처
- 전체 화면(다중 모니터 지원)
- **창 / UI 요소 자동 감지**: 창이나 인터페이스 요소(버튼, 패널, 웹 페이지 블록) 위에 마우스를 올리면 자동으로 윤곽이 표시되고 클릭 한 번으로 캡처. **마우스 휠**로 감지 수준 전환(요소 ⇄ 창)
- 드래그로 원하는 영역 선택. 크기 표시와 돋보기로 픽셀 단위 조정 가능

### 주석
캡처 후 커서 옆에 **아이콘 도구 모음**이 나타납니다(드래그로 이동 가능, 화면 가장자리 자동 회피):

- **마커** — 실선 스트로크. 굵기 / 색 / 투명도 조절 가능
- **형광펜** — 반투명 덧칠. 굵기 / 색 / 투명도 조절 가능
- **텍스트** — 글꼴 / 크기 / 색 / 스타일(일반, 굵게, 기울임, 취소선) 선택 가능. 상자 밖을 클릭하면 확정
- **도형** — 사각형 / 둥근 사각형 / 원. 테두리 또는 채우기, 선 굵기와 색 조절 가능
- **직선 / 화살표** — 드래그로 그리기. 양 끝의 화살표를 각각 켜고 끌 수 있으며, 굵기와 색 조절 가능. `Shift`로 45° 단위 스냅
- **이미지 붙여넣기** — PNG / JPEG / WebP 붙여넣기. 드래그로 이동, 휠로 크기 조절
- **흐림** — 가우시안 블러 / 모자이크. 강도 조절 가능
- **직접 선택 / 이동** — 배치된 주석 위에 마우스를 올려 **바로 드래그로 이동**. `Delete`로 삭제. 이동·삭제 모두 실행 취소 가능
- **색상 선택기** — Hex 입력, RGB, 투명도 지원(반투명 색은 체커보드 위에서 미리보기). 사용자 지정 색은 다음 실행에도 기억되며 우클릭으로 삭제 가능
- **실행 취소 / 다시 실행** — 버튼과 단축키 모두 지원. 모든 슬라이더에 실시간 숫자 표시

### 출력
- 클립보드로 복사
- PNG / JPG로 저장(다른 이름으로 저장 또는 기본 폴더로 빠른 저장)
- **화면에 고정**: 이미지가 항상 위에 떠 있습니다. 드래그로 이동, 휠로 크기 조절, 우클릭 메뉴. 여러 개 고정 시 Esc는 포커스된 것부터 닫고, 포커스가 없으면 모두 닫습니다

### 영역 녹화
- 전용 전역 단축키(기본 **F2**): 드래그로 녹화 영역을 선택해 시작, 다시 누르면(또는 중지 버튼) 종료
- 녹화 중 빨간 테두리로 영역 표시. 타이머와 중지 버튼이 있는 작은 바 제공
- 녹화 후 **편집기**가 열립니다: 반복 미리보기, 타임라인 트림 핸들(`Space` 재생, `←`/`→` 프레임 이동, `I`/`O`로 시작/끝 지점 설정), 진행률 표시와 함께 내보내기
- **GIF / MP4 / WebP**로 내보내기(편집기에서 바로 전환 가능). 설정에서 편집기를 건너뛰고 즉시 저장도 가능
- 마우스 커서 녹화 선택 가능, 프레임 속도 10–30 fps
- 번들된 `ffmpeg`로 인코딩. 별도 설치 불필요

### 기타
- 전역 단축키(**F1** 캡처, **F2** 녹화, 모두 사용자 지정 가능)와 **시스템 트레이** 아이콘에서 실행
- **다국어 지원**: 繁體中文 / English / 日本語 / 한국어 / Français / Deutsch / Español(기본은 시스템 언어)
- **라이트 / 다크 / 시스템 따르기** 테마. 모던한 라운드 UI와 다크 제목 표시줄
- 통합 **설정 창**: 언어, 단축키, 테마, 시작 시 실행, 저장 폴더, 녹화 형식 / 프레임 속도(단축키 칸은 키 조합을 누르기만 하면 설정)
- **Windows 시작 시 자동 실행**(선택)
- **자동 업데이트**: GitHub Releases로 새 버전 확인(설정에서 켜고 끄기, 수동 확인 가능). 클릭 한 번으로 다운로드 및 설치

## 설치

[Releases](https://github.com/taida957789/ScreenPaste/releases)에서 다운로드:

- **`ScreenPaste-<version>-setup.exe`** — 설치 버전(관리자 권한 불필요, `%APPDATA%\ScreenPaste`에 설치, 시작 메뉴 바로가기와 제거 프로그램 포함)
- **`ScreenPaste-<version>-win-x64-portable.zip`** — 무설치 포터블 버전

둘 다 자체 포함형으로 **별도의 .NET Runtime이 필요 없으며**, 영역 녹화용 `ffmpeg`도 번들되어 있습니다.

## 빠른 시작

![도구 모음](screenshots/ui.png)

1. 실행하면 시스템 트레이에 상주합니다. **F1**(또는 트레이 아이콘 더블클릭)로 캡처 시작.
2. 창이나 UI 요소 위에 마우스를 올려 자동 윤곽 표시 → 클릭으로 캡처, 또는 드래그로 영역 선택.
3. 팝업 도구 모음에서 도구를 고르고 매개변수를 조정해 선택 영역에 주석 작성.
4. **복사 / 저장 / 고정**으로 출력. `Esc`로 캡처 종료(주석이 있으면 먼저 확인하며, "다시 묻지 않음" 선택 가능).

녹화: **F2**를 누르고 드래그로 영역을 선택하면 녹화 시작. 다시 **F2**(또는 중지 버튼)로 종료한 뒤 편집기에서 미리보기·트림하고 GIF/MP4/WebP로 내보내기(설정에서 즉시 저장으로 변경 가능).

## 단축키 목록

| 상황 | 키 | 동작 |
|---|---|---|
| 전역 | `F1` | 캡처 시작(사용자 지정 가능) |
| 전역 | `F2` | 영역 녹화 시작 / 중지(사용자 지정 가능) |
| 영역 선택 중 | 휠 | 감지 수준 전환(UI 요소 ⇄ 창) |
| 주석 중 | `Ctrl+Z` / `Ctrl+Y` | 실행 취소 / 다시 실행(사용자 지정 가능) |
| 주석 중 | `Ctrl+C` | 클립보드로 복사(사용자 지정 가능) |
| 주석 중 | `Ctrl+S` / `Ctrl+Shift+S` | 다른 이름으로 저장 / 빠른 저장(사용자 지정 가능) |
| 주석 중 | `Delete` | 선택한 주석 삭제 |
| 주석 중 | `Shift`+드래그(직선) | 45° 단위 스냅 |
| 주석 중 | `Enter` 또는 상자 밖 클릭 | 텍스트 확정(`Shift+Enter` 줄바꿈) |
| 주석 중 | `Esc` | 선택 해제 → 캡처 종료(주석이 있으면 확인) |
| 녹화 편집기 | `Space` | 재생 / 일시정지 |
| 녹화 편집기 | `←` / `→` | 프레임 이동 |
| 녹화 편집기 | `Home` / `End` | 트림 시작 / 끝으로 이동 |
| 녹화 편집기 | `I` / `O` | 재생 위치에 트림 시작 / 끝 설정 |
| 고정 창 | `Ctrl+C` / `Esc` | 복사 / 닫기 |

설정(단축키, 도구 기본값, 테마, 시작 시 실행, 녹화 형식 / 프레임 속도 등)은 트레이 메뉴 → "설정…" 창에서 조정할 수 있습니다.

> 소스에서 빌드할 경우: `scripts/fetch-ffmpeg.ps1`을 한 번 실행해 번들용 `ffmpeg.exe`를 다운로드하세요(릴리스 시 CI에서 자동 실행됩니다).
