# ScreenPaste

軽快な Windows 用スクリーンショット・注釈ツール。ワンプッシュでキャプチャ、その場で注釈、画面にピン留めできます。

## 機能

### キャプチャ
- 画面全体（マルチモニター対応）
- **ウィンドウ自動認識**：マウスをウィンドウに重ねると自動で範囲を描き、クリックでキャプチャ
- ドラッグで範囲を自由選択。サイズ表示と、正確に境界を取れる拡大鏡付き

### 注釈
キャプチャ後、カーソルの横に**アイコンツールバー**がポップアップします（ドラッグで移動可能、画面端を自動で避けます）：

- **マーカー** — 実線のストローク。太さ / 色 / 不透明度を調整可能
- **蛍光ペン** — 半透明の重ね色。太さ / 色 / 不透明度を調整可能
- **テキスト** — フォント / サイズ / 色 / スタイル（標準・太字・斜体・打ち消し線）を選択可能
- **図形** — 四角形 / 角丸四角形 / 円。枠線または塗りつぶし。線の太さと色を調整可能
- **スタンプ** — PNG / JPEG / WebP 画像を貼り付け、ドラッグで移動、ホイールで拡大縮小
- **ぼかし** — ガウスぼかし / モザイク。ぼかしの強さを調整可能
- **カラーピッカー** — Hex 入力、RGB、不透明度の調整に対応
- **元に戻す / やり直し**

### 出力
- クリップボードにコピー
- PNG / JPG で保存（名前を付けて保存、または既定フォルダーへクイック保存）
- **画面にピン留め**：画像を最前面に浮かせ、ドラッグ・ホイールで拡大縮小・右クリックメニュー。複数ピン留め時は Esc でまずフォーカス中のものを閉じ、フォーカスがなければすべて閉じます

### その他
- グローバルホットキー（既定は **F1**）と**システムトレイ**アイコンからの起動
- **多言語対応**：繁体字中国語 / English / 日本語 / 한국어 / Français / Deutsch / Español（既定はシステムに追従）
- **ライト / ダーク / システムに追従**のテーマ
- 一元化された**設定ウィンドウ**：言語、ホットキー、テーマ、スタートアップ起動、保存フォルダー（ホットキー欄はキーの組み合わせを押すだけで設定可能）
- **スタートアップ時の自動起動**を設定可能
- **自動更新**：GitHub Releases から新バージョンを確認（設定で切り替え、または手動で今すぐ確認）。更新があれば確認のうえワンクリックでダウンロード・インストール

## インストール

[Releases](https://github.com/taida957789/ScreenPaste/releases) からダウンロード：

- **`ScreenPaste-<バージョン>-setup.exe`** — インストーラー版（管理者権限不要、`%APPDATA%\ScreenPaste` にインストール、スタートメニューのショートカットとアンインストーラー付き）
- **`ScreenPaste-<バージョン>-win-x64-portable.zip`** — インストール不要のポータブル版

どちらも自己完結型のため、**別途 .NET Runtime のインストールは不要**です。

## クイックスタート

![ツールバー](screenshots/ui.png)

1. 起動するとシステムトレイに常駐します。**F1**（またはトレイアイコンをダブルクリック）でキャプチャを開始。
2. ウィンドウにマウスを重ねて自動で枠を描きクリックでキャプチャ、またはドラッグで範囲を自由選択。
3. ポップアップしたツールバーでツールを選び、パラメーターを調整して、選択範囲に注釈します。
4. **コピー / 保存 / ピン留め** で出力。`Esc` で選択のやり直しに戻る、またはキャンセルできます。

設定（ホットキー、ツールの既定値、テーマ、スタートアップ起動など）は、トレイメニュー →「設定…」のウィンドウで調整できます。
